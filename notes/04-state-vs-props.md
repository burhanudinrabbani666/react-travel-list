## Controlled Elements

### State

- Internal Data, owned by compponent
- Component memory
- Can be updated by the components itself
- Updating state causes component to re-render

### Props

- External data, owned by parent Component
- similiar to function parameter
- Read Only
- Received new props causes component re-render. Usually when the parents state has been updated
